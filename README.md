# KorsDub site

Статический сайт с приветственным экраном первого посещения и главной страницей голосового проекта.

## GitHub Pages

Проект настроен для публикации через GitHub Pages workflow.

## Как работает

- Главная страница содержит hero-блок, меню, CTA, социальные ссылки и контактную почту.
- При первом визите посетитель видит welcome-экран.
- После нажатия кнопки «Войти на сайт» состояние сохраняется в `localStorage`.
- При следующих посещениях welcome-экран автоматически скрывается.

## Локальный запуск

Откройте `index.html` в браузере.
